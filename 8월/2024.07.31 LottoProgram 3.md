#### 4. jar파일로 getResource를 활용할 경우 생기는 문제
- jar파일 내부에서 getResource를 사용해서 file객체를 읽을 때 file 객체는 계층적 uri가 아니라서 예외 발생과 함께 에러가 난다.
- 해결 방법
	1) getResourceAsStream을 활용
	2) Path 경로를 활용
	3) FileReader( 상대 경로로 작성된 file객체나 상대경로 주소를 직접 입력)
- imageIcon은 괜찮다.
