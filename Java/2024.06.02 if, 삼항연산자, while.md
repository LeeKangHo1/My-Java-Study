
## if 조건문
- if (조건 1) {
	- 조건 1이 참일 때 실행하는 블록 }
- else if (조건 2) {
	- 조건1이 거짓이고 조건 2가 참일 때 실행하는 블록 }
- else {
	- 조건 1, 2 이 모두 거짓일 때 실행하는 블록 }
- 예제) 입력한 점수에 따라 다른 문장이 출력 되는 소스 코드

## 삼항연산자 (? :)
1. 삼항연산자는 간단한 if 연산자입니다.
2. (조건) ? 조건이 참일 때 : 조건이 거짓일 때 
3. if (a > b) max = a;
	-else max = b;
4.  max = (a>b) ? a : b
5. 3번과 4번은 같은 코드입니다.
