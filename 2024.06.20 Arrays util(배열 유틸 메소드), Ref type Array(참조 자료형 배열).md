### 이전 개념 복습
#### 정적 메소드(static method)
1. 인스턴스 생성 없이 호출 가능
	1. 인스턴스 생성 후 호출도 가능하지만 지양한다.
	2. main에서 바로 쓰기 좋다. 메소드 이름만으로 호출 가능하다.
2. 유틸리티 관련 함수를 만드는데 유용하게 사용된다.
